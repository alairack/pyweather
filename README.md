### Chinese
#### **介绍**
这是一个windows系统查询天气的工具。目前的窗口界面看着有点寒碜。
#### **功能**
- 自动获得ip地址
- 通过ip地址获得地理位置
- 自动查询天气
#### **运行**
1. 下载包含exe文件的压缩包.
2. 解压缩zip.
3. 打开解压后的文件夹，启动 "pyweather.exe".
#### **注意事项**
 目前暂不支持国外ip查询，如使用国外ip会报错！
 
**使用愉快**。
